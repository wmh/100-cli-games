# Day 2: Snake Game (貪吃蛇) 🐍

## 遊戲設計

### 核心機制
- 蛇由多個方塊組成，持續移動
- 吃到食物會變長，分數增加
- 撞到牆壁或自己會死亡
- 速度隨著長度增加而提升

### 遊戲畫面
```
╔══════════════════════════════════════════════════════════╗
║                                                          ║
║                          ●                               ║  食物
║                                                          ║
║                                                          ║
║              ▓▓▓▓▓▓                                     ║  蛇
║                                                          ║
║                                                          ║
╚══════════════════════════════════════════════════════════╝
Score: 50  Length: 8  Level: 2  (Arrow keys, Q=quit)
```

### 技術規格
- **畫面大小**: 60x25（與 Breakout 相同）
- **蛇身**: 用 `▓` 表示
- **蛇頭**: 用 `■` 表示
- **食物**: 用 `●` 表示
- **初始長度**: 3
- **初始速度**: 0.15 秒/幀
- **速度增長**: 每吃 5 個食物提速 10%

### 控制方式
- ⬆️ / W - 上
- ⬇️ / S - 下  
- ⬅️ / A - 左
- ➡️ / D - 右
- Q - 退出

### 遊戲規則
1. 蛇持續朝當前方向移動
2. 不能直接反向（例如向右時不能直接向左）
3. 吃到食物：長度 +1，分數 +10
4. 撞牆或撞自己：遊戲結束
5. 每 5 個食物：等級 +1，速度提升

### 計分系統
- 基礎分數：10 分/食物
- 等級獎勵：等級數 × 5
- 最終分數 = 食物數 × (10 + 等級獎勵)

### 特殊功能
- 暫停功能（空白鍵）
- 顯示最高分
- 速度等級顯示

## 實作要點

### 資料結構
```python
snake = [(x, y), (x-1, y), (x-2, y)]  # 頭到尾
direction = (1, 0)  # (dx, dy)
food = (fx, fy)
```

### 移動邏輯
```python
1. 計算新的頭部位置
2. 檢查碰撞（牆、自己）
3. 如果吃到食物：
   - 添加新頭部
   - 生成新食物
   - 增加分數
4. 否則：
   - 添加新頭部
   - 移除尾部
```
