# 磚塊位置修正 🧱

## 問題
右邊的磚塊超出遊戲邊界

## 原因分析

### 原始設計
```
遊戲寬度: 60
磚塊數量: 10 個/行
磚塊寬度: 5
間距: 1
計算: 10 * 6 + 2 (起始) = 62 ❌ 超出！
```

### 問題
```
最右邊磚塊: 
  起始位置: 9 * 6 + 2 = 56
  結束位置: 56 + 5 = 61
  邊框位置: 59
  結果: 超出 2 個字元！
```

## 解決方案

### 調整後設計
```
遊戲寬度: 60
磚塊數量: 9 個/行 (減少 1 個)
磚塊寬度: 5
間距: 1
總寬度: 9 * 5 + 8 * 1 = 53
起始位置: (60 - 53) / 2 = 3 (置中)
```

### 驗證
```
磚塊 1: x=3  → 結束於 7
磚塊 2: x=9  → 結束於 13
磚塊 3: x=15 → 結束於 19
磚塊 4: x=21 → 結束於 25
磚塊 5: x=27 → 結束於 31
磚塊 6: x=33 → 結束於 37
磚塊 7: x=39 → 結束於 43
磚塊 8: x=45 → 結束於 49
磚塊 9: x=51 → 結束於 55

右邊框: x=59
結果: ✅ 完美適配！
```

## 遊戲數據更新

- 總磚塊數: 50 → **45** (5行 × 9列)
- 每塊分數: 10 分
- 滿分: 500 → **450 分**

## 程式碼變更

```python
def init_bricks(self):
    # 原本: 10 個磚塊/行
    # 現在: 9 個磚塊/行，置中對齊
    
    brick_width = 5
    brick_spacing = 1
    total_bricks_per_row = 9  # ← 改為 9
    
    # 計算置中位置
    total_width = total_bricks_per_row * brick_width + (total_bricks_per_row - 1) * brick_spacing
    start_x = (self.width - total_width) // 2
```
