# Curses 升級完成！🎉

## 改進內容

### 🎯 主要改進
✅ **完全無閃爍** - 使用 curses 雙緩衝機制
✅ **流暢動畫** - 專業遊戲級渲染
✅ **豐富顏色** - 支援彩色磚塊和元素
✅ **更好控制** - 支援方向鍵和 A/D 鍵

### 📝 技術變更

#### 1. 核心渲染系統
**Before (os.system):**
- 每次完整清屏
- 閃爍嚴重
- 效能較差

**After (curses):**
- 雙緩衝渲染
- 完全流暢
- 高效能

#### 2. 顏色系統
```python
# 7 個顏色配對
1. RED    - 紅色磚塊
2. YELLOW - 黃色磚塊
3. GREEN  - 綠色磚塊
4. CYAN   - 青色磚塊
5. MAGENTA- 紫色磚塊
6. WHITE  - 球和板子
7. BLUE   - 保留使用
```

#### 3. 控制改進
- ⬅️ 左方向鍵 或 A
- ➡️ 右方向鍵 或 D
- Q 退出遊戲

#### 4. 新增工具模組
`utils/curses_helper.py` - 提供通用函數：
- `init_colors()` - 初始化顏色
- `setup_game_screen()` - 設置遊戲畫面
- `draw_box()` - 繪製邊框
- `draw_text_centered()` - 居中文字
- `show_message()` - 顯示訊息
- `check_terminal_size()` - 檢查終端大小

### 🔧 安裝需求

#### Linux / macOS
```bash
# curses 已內建，無需額外安裝
pip3 install rich colorama
```

#### Windows
```bash
# 需要安裝 windows-curses
pip3 install rich colorama windows-curses
```

或使用 requirements.txt（會自動處理平台差異）：
```bash
pip3 install -r requirements.txt
```

### 🐛 已知問題

1. **終端大小**
   - 需要至少 60x25 的終端視窗
   - 太小會顯示警告訊息

2. **表情符號**
   - 某些終端可能不支援 emoji
   - 遊戲主體使用 ASCII 字元，不影響遊玩

### 💡 提示

- 調整終端字體大小以獲得最佳體驗
- 使用全螢幕模式
- 確保終端支援 UTF-8 編碼
