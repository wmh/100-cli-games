### 🧠 核心邏輯

#### 球的物理
```python
1. 持續移動 (dx, dy)
2. 碰到上下牆反彈 (dy = -dy)
3. 碰到板子反彈 (dx = -dx)
4. 根據擊球位置調整角度
```

#### 板子碰撞
```python
# 擊球位置影響反彈角度
hit_pos = (ball_y - paddle_y) / paddle_height
ball_dy = (hit_pos - 0.5) * 2

上部 (0.0): dy = -1.0 (往上)
中部 (0.5): dy =  0.0 (水平)
下部 (1.0): dy =  1.0 (往下)
```

#### 計分系統
```python
球超出左邊界 → Player 2 得分
球超出右邊界 → Player 1 得分
達到 11 分 → 遊戲結束
```
