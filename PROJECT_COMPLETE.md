## 🚀 如何使用

### 安裝
```bash
git clone https://github.com/wmh/10-cli-games.git
cd 10-cli-games
pip install -r requirements.txt
```

### 執行
```bash
python main.py
```

### 單獨測試
```bash
python games/game_001_breakout.py
python games/game_002_snake.py
# ... 其他遊戲
```
